## Placeholders:

- Device:
	- `%versionize_device%`
- Locale:
	- Lowercase: `%versionize_locale%`
	- Uppercase: `%versionize_locale_upper%`
- Protocol Version
	- `%versionize_player_protocol_version%`
- Protocol ID
	- `%versionize_player_protocol_id%`
- Xbox username
	- `%versionize_username%`
- Xbox xuid
	- `%versionize_xuid%`
